  - NGÀY, THÁNG, NĂM, GIỜ, PHÚT, GIÂY
  - NỘI DUNG ĐÃ THAY ĐỔI
  - VIẾT LẠI TOÀN BỘ PROJECT DEFINITION BAO GỒM:
    - INPUT DATA
    - METHODOLOGY
    - EXPECTED RESULTS
  - VIẾT LẠI TOÀN BỘ FLOW HỆ THỐNG SAU KHI ĐÃ THAY ĐỔI
- VIẾT LẠI TOÀN BỘ CÁCH ĐỂ CHẠY CODE VÀO FILE `build.sh`
